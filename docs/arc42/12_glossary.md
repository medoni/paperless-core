# 12. Glossar

| Begriff | Definition |
|---------|------------|
| **PLC** | PaperlessCore - Projektname und Namespace-Prefix |
| **OCR** | Optical Character Recognition - Texterkennung aus Bildern/PDFs |
| **OTEL** | OpenTelemetry - Framework für Observability |
| **CQRS** | Command Query Responsibility Segregation - Trennung von Schreib- und Lesemodell |
| **ADR** | Architecture Decision Record - Dokumentation von Architekturentscheidungen |
| **IaC** | Infrastructure as Code - Infrastruktur-Definition als Code (Terraform) |
| **Clean Architecture** | Architekturmuster mit strikter Abhängigkeitshierarchie, Domain im Zentrum |
| **Repository Pattern** | Abstraktionsmuster für Datenzugriff |
| **Serverless** | Cloud-Computing-Modell ohne Server-Management (Functions as a Service) |
| **Document AI** | Google Cloud Service für OCR und Dokumentenverarbeitung |
| **Textract** | AWS Service für OCR und Dokumentenverarbeitung |
| **Firestore** | Google Cloud NoSQL-Datenbank |
| **DynamoDB** | AWS NoSQL-Datenbank |
